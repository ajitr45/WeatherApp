# Weather App

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript** that fetches real-time weather data from the **WeatherAPI**.

# Features

*  Search weather by city name
*  Display current temperature
*  Dynamic weather condition icon
*  Show humidity percentage
*  Display wind speed
*  Show cloud coverage
*  Responsive and modern user interface
*  Handles invalid city names gracefully

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* WeatherAPI
